# News Headlines Filler (Figma plugin)

Плагин ищет текстовые слои с именем вида `Новость 1`, `Новость 2`, … и подставляет в них актуальные заголовки без повторов.

## Возможности
- Источник заголовков: ручной ввод (по строкам) или URL (JSON-массив строк/объект с `headlines`/`items`/`titles`/`articles`, либо обычный текстовый ответ).
- Предпросмотр: кнопка «Показать список» собирает уникальные заголовки, убирает дубли с макетом и показывает, сколько строк будет использовано.
- Заполнение: по номеру `Новость N` или по порядку слоёв; можно переименовать слои в текст заголовка или по шаблону `Заголовок {n}` / `… {text}`.
- Защита от дублей: новые строки сравниваются в нижнем регистре и с нормализацией пробелов; существующие тексты слоёв исключаются.

## Как запустить в Figma
1. Откройте Figma → `Plugins` → `Development` → `Import plugin from manifest…`.
2. Выберите файл `figma-plugin-news-headlines/manifest.json`.
3. Запустите плагин через `Plugins` → `Development` → `News Headlines Filler` (или через кнопку релонча на слоях).

## Быстрый сценарий
1) Оставьте «Вручную», вставьте заголовки построчно.  
2) Нажмите «Показать список» — увидите уникальные строки и сколько слоёв на странице.  
3) «Заменить в макете» — плагин заполнит слои и, при необходимости, переименует их.

## Советы
- Чтобы брать данные из вашего API, отдавайте `["заголовок 1", "заголовок 2"]` или `{ "headlines": [...] }`. Любой текстовый ответ тоже подойдёт — строки разделяются переводом строки.
- Префикс/суффикс применяются перед дедупликацией, поэтому одинаковые строки с разными префиксами считаются разными.
- Если слои пустые и Figma ругается на шрифты, задайте им базовый шрифт заранее (или оставьте хотя бы один символ, чтобы плагин подгрузил семейство).
